# FocusLock 🔒
**Discipline launcher by CoreviBS**

## Build APK from Mobile (No PC needed)

### Step 1 — Upload to GitHub (from phone)
1. Open **github.com** in Chrome
2. Sign in / create free account
3. Tap **+** → **New repository**
4. Name it `focuslock`, set to Private, tap **Create**
5. Tap **uploading an existing file**
6. Upload ALL files from this folder (maintain folder structure)
   - Or use **GitHub Mobile app** → easier drag & drop

### Step 2 — Trigger the build
1. On GitHub, go to **Actions** tab
2. You'll see "Build FocusLock APK" workflow
3. It runs automatically after upload. Or tap **Run workflow**
4. Wait ~5 minutes ⏳

### Step 3 — Download APK
1. Click the completed workflow run
2. Scroll to **Artifacts** section
3. Download **FocusLock-APK**
4. Unzip → get `app-release.apk`
5. Install on phone (allow unknown sources in settings)

### First Run on Phone
1. Open FocusLock
2. Grant **Usage Access** permission
3. Enable **Accessibility Service**
4. Set as **Default Launcher** when Android asks

## App Features
- 📱 Full launcher (home button = FocusLock, no escape)
- ⏱️ Instagram: 1hr/day limit
- ▶️ YouTube: 3hr/day limit  
- ⚡ Strike system: 3 strikes = 1hr full lockdown
- 🌙 Strikes reset at midnight
- 🔒 BlockedActivity kills back button — no bypass
